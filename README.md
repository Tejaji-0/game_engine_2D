# 2D Java Game Engine

A simple 2D game engine built with Java and AWT/Swing.

## Project Structure
- `src/` - Source code
- `assets/` - Game assets (sprites, maps, etc.)
- `docs/` - Documentation

## Status
- [x] Basic project setup
- [ ] Window creation
- [ ] Game loop
- [ ] Rendering
- [ ] Input handling
- [ ] Sprite loading
- [ ] Entity system
- [ ] Scene management
- [ ] Collision detection
- [ ] Tilemap support

## Build & Run
```bash
javac -d bin src/**/*.java
java -cp bin engine.Main
```
